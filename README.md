# 🌱 Salve o Planeta

## Projeto Agrinho 2026 – Programação

### 📖 Sobre o Projeto

**Salve o Planeta** é um jogo educativo desenvolvido para o Concurso Agrinho 2026, com o objetivo de conscientizar crianças e jovens sobre a importância da preservação ambiental e das ações sustentáveis para a construção de um futuro melhor.

O jogador assume o papel de um herói ambiental que deve coletar itens positivos para o meio ambiente, evitar fontes de poluição e cumprir objetivos para avançar pelas fases, onde o jogador planta árvores desbloqueando novas fases e cumprindo missões sustentáveis. As recompensas conquistadas podem ser utilizadas para desenvolver e melhorar o próprio jardim sustentável.

O projeto busca demonstrar que pequenas atitudes podem gerar grandes transformações, fortalecendo a relação entre sociedade, meio ambiente, produção de alimentos e qualidade de vida.

## 🎯 Objetivos

* Promover a conscientização ambiental por meio da tecnologia e da gamificação.
* Incentivar a adoção de práticas sustentáveis no dia a dia.
* Demonstrar como pequenas ações podem contribuir para a preservação do meio ambiente.
* Desenvolver o pensamento crítico sobre os impactos da poluição e do descarte inadequado de resíduos.
* Estimular a responsabilidade socioambiental e o cuidado com os recursos naturais.
* Relacionar a preservação ambiental ao tema do Agrinho 2026: "Agro forte, futuro sustentável: Equilíbrio entre produção e meio ambiente".
* Utilizar a programação como ferramenta de educação, aprendizagem e transformação social.

## 🌾 Relação com o Tema Agrinho 2026

O projeto está alinhado ao tema **""Agro forte, futuro sustentável: Equilíbrio entre produção e meio ambiente"."**, destacando a importância da preservação ambiental e da responsabilidade coletiva na construção de uma sociedade mais sustentável.

Por meio do jogo, os participantes aprendem sobre:
* Preservação da natureza;
* Sustentabilidade;
* Reciclagem;
* Combate à poluição;
* Consumo consciente;
* Responsabilidade socioambiental.

## 🎮 Como Jogar

### Computador

*Movimente o cursor do mouse para controlar o personagem.
*Colete os itens positivos para ganhar pontos e moedas.
*Evite os elementos poluentes para não perder vidas.
*Complete os objetivos de cada fase para avançar no jogo.
*Utilize os botões da interface para pausar ou acessar o jardim.
*Em seu jardim, utilize suas moedas para cumprir as missões sustentáveis.


### Celular

* Toque e arraste o personagem para movimentá-lo.
*Colete os itens positivos para ganhar pontos e moedas.
*Evite os elementos poluentes para não perder vidas.
*Complete os objetivos de cada fase para avançar no jogo.
*Utilize os botões da interface para pausar ou acessar o jardim.
*Em seu jardim, utilize suas moedas para cumprir as missões sustentáveis.
